## 2. VISUAL & IMAGE GENERATION INSTRUCTIONS

### 2.1 Tone and Style
- **Aesthetic**: Bright, modern, professional, photorealistic
- **Photography Style**: AI-real, high-quality, natural lighting
- **Mood**: Energetic, trustworthy, innovative, African-focused
- **Composition**: Dynamic, asymmetric, human-centered
- **Forbidden Colors**: #00043F (navy) - NO DARK NAVY BACKGROUNDS

### 2.2 Color Palette Application Rules
- **Primary Accent**: #f8cf40 (gold) - Use for CTAs, highlights, key elements
- **Secondary Neutral**: #8A8D91 (gray) - Use for text, borders, subtle backgrounds
- **Primary Background**: #ffffff (white) - Clean, bright, minimal
- **Text**: #000000 (black) - Body copy, high contrast
- **Avoid**: #00043F (navy) - Replace with white or light gray backgrounds

| Image Type | Dimensions | Aspect Ratio | Format | Use |
